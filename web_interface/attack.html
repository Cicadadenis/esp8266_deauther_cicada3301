Deauth
Цели: 0
Пак/с: 0/0
Отправляет фреймы деаутентификации для отключения выбранных WiFi устройств.
Работает т.к. большинство устройств не учитывают защиту 802.11w.
Внимание: Только одна цель! Несколько целей на разных каналах = невозможно переподключиться.
Beacon
Цели: 0
Пак/с: 0/0
Засыпает эфир поддельными Маячковыми фреймами → создаются призрачные WiFi-сети.
Имена сетей берётся из списка SSID.
Хорошо для тестирования видимости сетей.
Probe
Цели: 0
Пак/с: 0/0
Отправляет Probe-запросы для SSID из списка.

На домашней сети видимый эффект минимален.
