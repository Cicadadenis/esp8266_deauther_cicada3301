Как смотреть клиентов
Авто (фон): после «Скан Wi-Fi сети» и «Обновить» клиенты появляются сами (promiscuous, ~2 с). В процессе устройство может переключать радио по каналам — иногда веб на секунды «подвисает». Подождите 10–20 с и снова «Обновить».
Глубокий скан: ① Скан Wi-Fi сети → ② выберите одну Wi-Fi сети → ③ Скан клиентов (15–45 с, deauth + sniff) → ④ Обновить. Во время скана радио фиксируется на канале цели — SoftAP/веб могут временно отключиться, это нормально.
Кнопки: «Клиент.» на карточке Wi-Fi сети — быстрый скан клиентов этой сети. «HS» — handshake (лучше на конкретном клиенте). EEPROM — сохранить Wi-Fi сети/канал в настройки. Клон SSID — на вкладке SSIDs.
Если станций 0: сначала есть Wi-Fi сети в списке? клиенты привязаны к известным BSSID. Попробуйте «Скан клиентов» 30–45 с на канале Wi-Fi сети. PMF/WPA3 мешают.
Только своя лабораторная Wi-Fi сети! Скан Wi-Fi сети → Обновить → ждите клиентов. На время скана точка ESP может пропасть.
